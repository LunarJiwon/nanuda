# 나누다 (Nanuda)

A personal/small-team blog with four content rooms — 일상 (daily), 정보 (info), 예술 (art), 글귀
(quote) — built with Next.js (App Router, TypeScript, Tailwind CSS) and Firebase
(Auth/Firestore/Storage), implemented from a Claude Design handoff in `design/`.

**See [SETUP.md](./SETUP.md)** for required manual Firebase Console steps, how to seed sample
content, how to run the dev server, and documented scope decisions.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
